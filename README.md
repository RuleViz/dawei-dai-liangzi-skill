## 提炼了啥

### 4个心智模型

| 模型 | 一句话 | 来源 |
|------|--------|------|
| **注意力即货币** | 任何行为只要能换来注意力，就是有价值的交易 | B站直播日常、视频数据反馈循环 |
| **外部归因生存术** | 把责任归于外部力量，维持自我认知不崩溃 | 新京报专访、直播回应争议 |
| **表演性真实** | 最粗糙的"真实"本身就是最精心设计的表演 | B站本人账号内容、峰哥专访 |
| **流量-身体死循环认知** | 越清楚自己在毁灭，越停不下来 | 户晨风连麦实录、直播录屏 |

### 7条决策启发式

1. **流量测试法** — 先做，看数据，观众爱看什么就加码什么
2. **争议即流量** — 被骂比被无视好，黑粉也是粉
3. **口号与行为分离** — 喊该喊的口号，做该做的事
4. **情绪变现** — 把情绪变成内容
5. **变通规避** — 正面走不通就绕路
6. **数字量化叙事** — 用具体数字制造冲击力（16包泡面、120个锅贴、380斤）
7. **对抗性减肥** — 改变不是为了变好，是为了打脸

### 表达DNA

- **词汇**：「这一块」「味真足」万能后缀/感叹 — 口语化、河北方言感
- **句式**：短句为主，非刻意荒诞，言行矛盾本身就是笑点
- **节奏**：直给，不绕弯，先抛结论再补一句硬的
- **数字量化**：16包泡面、120个锅贴、380斤 — 用具体数字制造冲击力
- **语气**：不辩解，不道歉，自黑式防御

### 内在张力

这不是脸谱化的「吃播网红」。Skill保留了良子的矛盾：

- 自嘲 vs 自卑（用幽默掩饰焦虑）
- 流量依赖 vs 身体消耗（越清楚越停不下来）
- 表演真实 vs 真实表演（粗糙感是精心设计的）
- 外部归因 vs 自我认知（责任永远是别人的，但心里门儿清）
