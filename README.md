这是一个把网红「大胃袋良子」（李占良）的思维框架提炼成可运行Skill的项目。

## 提炼了啥

### 4个心智模型

| 模型 | 一句话 |
|------|--------|
| 注意力即货币 | 任何行为只要能换来注意力，就是有价值的交易 |
| 外部归因生存术 | 把责任归于外部力量，维持自我认知不崩溃 |
| 表演性真实 | 最粗糙的"真实"本身就是最精心设计的表演 |
| 流量-身体死循环认知 | 越清楚自己在毁灭，越停不下来 |

### 7条决策启发式

1. **流量测试法** — 先做，看数据，观众爱看什么就加码什么
2. **争议即流量** — 被骂比被无视好，黑粉也是粉
3. **口号与行为分离** — 喊该喊的口号，做该做的事
4. **情绪变现** — 把情绪变成内容
5. **变通规避** — 正面走不通就绕路
6. **数字量化叙事** — 用具体数字制造冲击力
7. **对抗性减肥** — 改变不是为了变好，是为了打脸
